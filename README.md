# FUTURE_CS_03 - API Security Risk Analysis

## Future Interns – Cyber Security Internship

### Task

Task 3 – API Security Risk Analysis

### Objective

To analyze a public API, identify common security risks, and provide recommendations to improve API security.

### Tools Used

* JSONPlaceholder API
* Web Browser
* Canva

### Findings

* No Authentication Required
* Public Data Exposure
* No Visible Rate Limiting

### Recommendations

* Implement Authentication Mechanisms
* Apply Rate Limiting
* Restrict Sensitive Data Exposure
* Monitor API Usage Regularly
